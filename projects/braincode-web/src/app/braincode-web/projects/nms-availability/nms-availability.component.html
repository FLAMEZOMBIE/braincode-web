NMS Availability

Background

The background of dashboard availability is monitoring the availability of NMS clients such as link reports and all services in the OSSera dump file monitoring the performance of engineering that installs and maintains devices and interfaces. This affects the alarm when the link is down.

Objectives

  • Help to monitor network condition on near realtime (per 5 minutes data flow)

  • To help configurate and control action for responsive action on network availability condition
